# Krakow Frame

## Описание
Krakow Frame — веб-приложение, созданное на базе **React**, **TypeScript** и сборщика **Vite**. Исходный код приложения находится в папке `app`.

## Как запустить проект локально

### Шаг 1. Перейдите в папку с приложением
Все конфигурационные файлы и исходники находятся в папке `app`, поэтому сначала нужно перейти в неё:
```bash
cd app
```

### Шаг 2. Установите зависимости
При первом запуске (или после обновления пакетов) установите все необходимые зависимости с помощью npm:
```bash
npm install
```

### Шаг 3. Запустите сервер для разработки
Для запуска локального сервера в режиме разработки выполните:
```bash
npm run dev
```

Обычно после этого проект становится доступен по адресу **http://localhost:5173**. Любые изменения в коде будут автоматически применяться в браузере (Hot Module Replacement).

---

## Дополнительные команды

Все эти команды также должны выполняться из папки `app`:

- `npm run build` — Сборка проекта для production (готовые оптимизированные файлы появятся в папке `app/dist`).
- `npm run preview` — Локальный предпросмотр уже собранного проекта (позволяет проверить, как проект будет работать на реальном сервере).
- `npm run lint` — Запуск проверки качества кода с помощью ESLint.
